# Bilingual Answering Chatbot 🌍🤖
## Project Description
This project is a bilingual answering chatbot that extracts information from websites to answer user questions. The chatbot is not creative and can only respond to questions related to the scraped content.

### Key Features
Web Scraping: Extracts data from a given website to answer questions

Bilingual Support: Answers in both Arabic and English based on the user's initial language

NLTK Integration: Uses Natural Language Processing for question answering

Language Detection: Automatically detects conversation language from first interaction

### Supported Languages
English 

Arabic 

## Installation & Usage
1- Clone this repository

2- Create a folder named static in the project directory

3- Run the code

### Configuration
By default, the chatbot works with Wikipedia pages about Lebanon in both Arabic and English. You can modify the URL to scrape any other webpage

### Limitations
- The chatbot can only answer questions based on the scraped content

- Requires static folder for proper operation

- Currently configured for Wikipedia but can be adapted to other sites
